## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/js-programs.git
   ```
2. Navigate to the program directory:
   ```bash
   cd js-programs
   ```
3. Run a JavaScript file using Node.js:
   ```bash
   node script.js
   ```
   Or, open it in a browser console for execution.
